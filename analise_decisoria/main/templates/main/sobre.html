Quem Somos?!

Alunos do sétimo período de sistemas de informação, que através da disciplina de Tópicos Avançados desenvolveram um site para o auxílio de tomadas de decisões.

Equipe de desenvolvimento do site.

O Projeto!

Utilizando métodos para tomadas de decisão das escolas francesas e americana para desenvolver o algoritmo oferecendo opções para o usuário para a tomada de decisões.

Na escola francesa foram utilizados os métodos de Electre I, Electre III, Electre Tri, Soft Electre Tri.

Na escola americana foi utilizado o método AHP.
